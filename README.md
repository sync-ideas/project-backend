# ASISTENCIAS BACKEND 
## CREATE
- npm install --save-dev typescript @types/express @types/node ts-node-dev
- npx tsc --init
- npx ts-node-dev src/index.ts
- npm install prisma @prisma/client --save-dev
- npx prisma init
- npx prisma db push
- npx prisma migrate dev --name init
